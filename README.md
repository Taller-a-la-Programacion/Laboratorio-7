## convertirDecHex(num)
La función debe recibir un parámetro **entero** positivo **mayor** o **igual** a **cero** y convertirlo a representación **hexadecimal**
La salida será en formato string

```python
>>>convertirDecHex(5)
'5'
>>>convertirDecHex(10)
'A'
>>>convertirDecHex(25)
'19'
>>>convertirDecHex(30)
'1E'
>>>convertirDecHex(300)
'12C'
```
